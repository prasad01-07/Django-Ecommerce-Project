

# Django Ecommerce Project

A modern Ecommerce Web Application built using Django and SQLite. This project allows users to browse products, register accounts, manage profiles, add products to cart, and place orders through an intuitive interface.

**Website:** https://django-ecommerce-project-rus1.onrender.com

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/prasad01-07/Django-Ecommerce-Project.git
cd Django-Ecommerce-Project
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Run Development Server

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
